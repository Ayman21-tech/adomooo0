# Adomo AI
**Smart Education & Exam Preparation Platform for Play Group → Class 9 (Bangladesh-first)**

Adomo AI is a modern learning platform designed for students in Bangladesh. It supports **Bangla + English**, helps students study selected subjects, upload book pages, prepare for exams, track progress, and improve weaknesses with a clean, fast UI.

> **Status:** V3 (Stage 1) — Main features working, UI/UX polishing ongoing.

---

## ✨ Key Features
- **Play Group to Class 9 support**
- **Subject-based learning dashboard**
- **Preparation hub** (Notes, Revision, Abbreviations/Full Form, Practice, etc.)
- **Book/Syllabus upload system** (organized by Subject → Chapter → Pages)
- **Progress tracking** (weakness + consistency focus)
- **Bangla + English language switching**
- **Light/Dark mode**
- **Smooth UI transitions and modern layout**
- **Settings personalization** (class, subjects, language, preferences)

---

## 🧭 How It Works (User Flow)
1. Start → Sign in
2. Choose class
3. Enter school name
4. Select subjects
5. Choose medium (Bangla Medium / English Version / English Medium)
6. Home dashboard with **5 main tabs**:
   - Progress
   - Subjects & Learn
   - Upload Books
   - Preparation/Exam
   - Settings
